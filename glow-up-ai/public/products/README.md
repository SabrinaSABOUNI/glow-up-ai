Dépose ici les vraies photos produit, nommées `<id>.jpg` (l'id est celui du
produit dans src/catalog.js, ex. `lrp_effaclarduo.jpg`), puis ajoute
`image: "/products/lrp_effaclarduo.jpg"` sur le produit correspondant dans
catalog.js. Sans ce champ, une icône de remplacement s'affiche automatiquement.
